terraform/
This is your IaC engine room.
Each file has a clear purpose:

main.tf — core resources (VPCs, subnets, IAM, EC2, S3, etc.)

variables.tf — input variables for customization

outputs.tf — values Terraform should print after deployment

provider.tf — AWS provider, region, backend config, etc.

This is exactly how Terraform modules and root configurations are structured.
